#  iREPLY Inventory Management System Documentation

Everything you need to install, configure, operate, and maintain the Inventory Management System — written for developers and end users.

## What is this?

The iREPLY Inventory Management System is a modern, full-stack web application built with Laravel 12 (Backend) and React 18 (Frontend), designed for comprehensive equipment and asset management. This documentation covers everything from first install to advanced maintenance.
